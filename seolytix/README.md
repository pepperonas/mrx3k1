# seolytix

Eine moderne Anwendung zur Analyse der SEO-Stärke von Webseiten.

## Projektstruktur

- `/frontend` - React-Frontend
- `/backend` - Express-Backend

## Installation

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend

```bash
cd backend
npm install
npm start
```

## Entwicklung

- Frontend läuft auf http://localhost:3000
- Backend-API läuft auf http://localhost:5010
